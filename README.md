# Twemoji

### Requirements

* NodeJS, npm / yarn
* MongoDB
* nodemon

### Installation

```sh
$ git clone https://github.com/hkaden/Twemoji.git
$ cd Twemoji-master
$ cp config.example.js config.js
# Edit config.js
$ yarn install
```

#### Dev env
```sh
$ yarn run dev
```

### Build
```sh
$ yarn run build
```
